# Data Sources & Methodology

## Methodology

All data sourced exclusively from official government MoD statements, verified news agencies, and established conflict monitoring organisations. No estimates or projections are used. Missing data is left blank rather than interpolated.

## Data Policy

- **No Estimates**: We never estimate or project data. If official sources have not published a figure, it is left blank.
- **Verification Threshold**: Data is only added when confirmed by at least one official government source or two independent verified news sources.
- **Update Frequency**: Data updated daily via automated MoD account monitoring + manual review.
- **Open Source**: Full source code and raw data available at [github.com/takahser/uae-dashboard](https://github.com/takahser/uae-dashboard).

### Notes

- **UAE**: MoD suspended publishing detected/intercepted breakdown after Mar 12/13 for OPSEC. Only "engaged" totals published since then. Mar 13+ daily entries left blank per no-estimates policy.
- **Israel**: IDF does not publish daily intercept breakdowns. Only total strike count confirmed publicly. Daily detail unavailable.

## How to Report Incorrect Data

If you believe any data point is incorrect or a source has been misattributed, please [open a GitHub issue](https://github.com/takahser/uae-dashboard/issues/new) with:

1. The specific data point in question
2. The correct figure (with source link)
3. Why the current data is incorrect

We take data accuracy seriously and will review all reports promptly.
